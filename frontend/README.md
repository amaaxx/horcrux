# Horcrux

> **Digital artifacts and complex systems.** > A high-performance, atmospheric portfolio engine built to house the "vessels" of my technical craft.

##  The Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Motion:** Framer Motion + GSAP
- **Typography:** Satoshi / Geist Mono

##  The Vessels
1. **Ground Truth Engine:** RAG architecture for deterministic AI accuracy.
2. **Strata:** Recursive folder-logic note system.
3. **BLW Portal:** Enterprise-grade intranet deployment.

##  Technical Flex
This repository utilizes the **Model Context Protocol (MCP)** and **Tailwind v4's** new `@theme` engine for absolute design control and LLM-readability.
